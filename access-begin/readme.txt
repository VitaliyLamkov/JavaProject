Система  для учета компьютеров позволяет вести инвентаризационный учет компьютеров, оборудования на предприятии, офисной техники, принтеров. Клиент переделан с использование Java EE и java DAO, в основном для изучения и праработки данной темы. База используется Access 2007, для соместимости со старым клиентом.
